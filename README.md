## 概要
初めて作成した個人アプリです！まだまだ、作成途中ですが随時更新していきます！<br>
このアプリでは、前職の小売業で培った商品知識を活かし、**「筋肉に良い商品」**をピックアップした主に三大栄養素の詳細が確認できるアプリになっています。

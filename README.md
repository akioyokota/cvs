## 概要 ⛵️
初めて作成した個人アプリ**CVS**です！<br>
このアプリを作ろうと思った理由は、筋トレが好きなのと前職の小売業で培った商品知識を活かしたいと思ったのがきっかけです。<br>
こちらのアプリでは**筋肉に良い商品**を私の独断と偏見でピックアップし、主に三大栄養素の詳細が確認できるアプリにする予定です。
